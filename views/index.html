Custom Coub API

USAGE

  1. To get results, do

    GET /hot/<category>/

    where category may be:

      • all
      • animals-pets
      • mashup
      • anime
      • movies
      • gaming
      • cartoons
      • art
      • music
      • news
      • sports
      • science-technology
      • celebrity
      • nature-travel
      • fashion
      • dance
      • cars
      • nsfw

  2. You also can filter results by count, duration and period

    GET /hot/all/?duration=228&period=weekly

    where

    • count is how many videos you will receive. It must be an integer value
    • duration is how much time (in seconds) the final video will long. It must be an integer value. Default value is 610
    • period is the period of which selection is made. May be weekly, monthly, quarter or half. Empty value is daily as default.

    NOTE: In case if you set count and duration, only duration will use.
